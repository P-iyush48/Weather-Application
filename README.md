# Weather-Application
Here you can search any country Weather information. 
See what is the current temperature that country or a city.

Api used from : WeatherApi (https://www.weatherapi.com/ )

Api Url: http://api.weatherapi.com/v1/forecast.json?key={your-access-key}&q={location}&aqi=yes;

**this url returns the full day weather information.

This Application deploy in the Netlify { https://weather-application-piyush48.netlify.app/ }
